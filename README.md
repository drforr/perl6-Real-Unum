NAME
====

Real::Unum - Implementation of Posits from posithub.org

SYNOPSIS
========

```perl6
use Real::Unum;
```

DESCRIPTION
===========

Real::Unum is an extension of real/floating point that claims to be more accurate and faster than POSIX reals.

ORIGINAL TEXT
=============

https://posithub.org/docs/Posits4.pdf
